<!--- Overview--->
This is a full-stack portfolio web application developed using React.js, Node.js, Express.js, and MongoDB.
The application allows users to submit their contact details through a web form, which are then stored securely in a database. It is designed to demonstrate full-stack development skills and backend data handling.
<!---Technology Stack--->
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Tools: Git, GitHub, VS Code
